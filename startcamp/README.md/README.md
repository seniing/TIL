# Today I Learned

- `mkdir` : 폴더 생성
- `cd` : 폴더 이동
  - `cd ..` : 상위 폴더 이동
  - `.` : 현재폴더 , `..` : 상위폴더
- `touch` : 파일 생성
- `rm -r (name)` : 삭제
  - `rm -rf (name)`
- `ls` : list
  - `ls -a` : 숨김폴더 포함



- git 명령어

  ```
  $ git add .

  $ git push -u origin master
  ```

  - `git status`
